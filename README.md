# 🌿 GreenNest - AI-Powered Plant Disease Detection
![image](https://github.com/user-attachments/assets/7ee888e9-51eb-4235-8d83-551e8d7cb21c)

 
GreenNest is an intelligent plant care system that helps users identify plant diseases and provides treatment recommendations using artificial intelligence. Our solution makes plant healthcare accessible to everyone, from home gardeners to agricultural professionals.

## ✨ Features

- 🔍 Real-time plant disease detection
- 💊 Personalized treatment recommendations
- 📱 Mobile-friendly interface
- 📊 Disease tracking and monitoring
- 🌱 Plant care guides and tips
- 📈 Progress tracking and history

## 🚀 Getting Started

### Prerequisites

- Node.js v16 or higher
- Python 3.8+
- TensorFlow 2.x
-  MongoDB

### Installation
```
cd frontend
npm install

cd backend
npm install
```
### Starting out
Run these commands seperately
```
cd frontend && npm run dev

cd backend && node server.js
```
## 🏗️ Architecture

Frontend: React.js with VueJS
Backend: Python FastAPI
Database: PostgreSQL
AI Model: TensorFlow/PyTorch
Image Processing: OpenCV

📱 Usage

Upload or take a photo of your plant
Wait for AI analysis
Get detailed disease information and treatment plan
Track treatment progress
Access care recommendations

🛠️ Tech Stack

Frontend: React.js, TailwindCSS, Redux
Backend: Python, FastAPI
Database: PostgreSQL
AI/ML: TensorFlow, OpenCV
Cloud: AWS/GCP
DevOps: Docker, GitHub Actions
